How and Why to Use Excel File Data Input in Cladevo

Excel data input is designed specifically for datasets derived from molecular marker research, such as PCR and gel electrophoresis results. These methods are essential for identifying genetic variations, analyzing DNA sequences, and understanding the genetic makeup of samples. PCR amplifies specific DNA segments, while gel electrophoresis separates these fragments based on size, making it easier to visualize and interpret results.

If you’re working with data from these techniques, you’ll learn how to structure your Excel files properly and when this functionality is most useful. I’ll focus on the practical steps and share insights into the technical requirements, making sure you’re set up for success in your analyses.
