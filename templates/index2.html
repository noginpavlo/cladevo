Welcome to
Cladevo

A free website where you can build phylogenetic trees just like that

Cladevo is a web-based tool for creating dendrograms using UPGMA and Neighbor-Joining hierarchical clustering methods. It features a user-friendly interface, supports FASTA files, and allows manual sequence input—making it an accessible and efficient solution for research and learning without the need for complex software.

Cladevo features

Flexible data input

You can build a dendrogram using FASTA files containing two or more sequences. Alternatively, you may input sequences manually or use Excel files for data input. For guidance on formatting your data in Excel, please refer to the provided instructions here.

Complete automation

The calculation process is fully automated and requires no additional action beyond data input. Steps such as sequence alignment, dissimilarity matrix calculation, dendrogram construction, and visualization are entirely automated, eliminating the need for user intervention.

Cladevo is free

Cladevo is completely free to use, offering full access to all features without requiring a subscription or user authentication. It is entirely ad-free, ensuring a distraction-free experience for users. Whether you're a student exploring clustering methods or a researcher seeking quick data insights, Cladevo provides a efficient and accessible tool for sequence analysis.

Contains educational materials

Cladevo provides all the educational resources necessary to use the application effectively and more. Comprehensive video tutorials are available in the How to Use section. Additionally, the Theory section offers an overview of the foundational concepts behind the UPGMA (Unweighted Pair Group Method with Arithmetic Mean) and Neighbor-Joining algorithms.
